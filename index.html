What is web development?
Web development is simply using html and css to create a website. Any website you see was designed, tested, and refined by a web developer. Without web development and web developers, any website you use would not exist and the internet would be nowhere near where it is today.

Becoming a Front-End Web Developer Characteristics of a Front End Web Developer
Front-end web developing is one of the easiest ways to get your feet wet when it comes to coding. It has the easiest learning curve, and takes minimal practice to become proficient at it. This career is done all around the world requiring only a computer. The industry is constantly growing because as more companies and people switch to online, the more websites are needed. The characteristincs of a front-end web developer is that one must be motivated, good at problem solving, and proficient at both understanding and applying a solution to a problem. Many who try to learn how to code end up quitting in the first week because they do not persevere through the hardships they encounter.
